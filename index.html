Etiquetas HTML

  1. <!DOCTYPE html>

    Descripción: Declara el tipo de documento y la versión de HTML que se está utilizando. En HTML5, la declaración es simple y se coloca en la primera línea del documento.

    Ejemplo de uso:

    <!DOCTYPE html>

    Información adicional: Esta declaración no es una etiqueta HTML, pero es esencial para que los navegadores interpreten correctamente el documento como HTML5.


  2. <html>

    Descripción: Define el inicio y el final de un documento HTML. Es el elemento raíz que contiene todo el contenido del documento, incluyendo los elementos <head> y <body>.

    Ejemplo de uso:

    <html>
<head>
<meta charset="UTF-8">
<title>Título del Documento</title>
</head>
<body>
<h1>Hola Mundo</h1>
</body>
</html>

    Información adicional: El atributo lang se usa para especificar el idioma del documento.


  3. <head>

    Descripción: Contiene metadatos (datos sobre datos) del documento, como el título, enlaces a hojas de estilo, y scripts.

    Ejemplo de uso:

    <head>
    <meta charset="UTF-8">
    <title>Título del Documento</title>
</head>

    Información adicional: El elemento <head> no se muestra en la página web, pero es esencial para configurar el documento HTML.


  4. <title>

    Descripción: Define el título del documento que se muestra en la barra de título del navegador o en la pestaña.

    Ejemplo de uso:

    <head>
    <title>Título de la Página</title>
</head>

    Información adicional: El contenido de <title> debe ser breve y descriptivo para ayudar a los usuarios a identificar la página.


  5. <link>

    Descripción: Define una relación entre el documento actual y un recurso externo, comúnmente utilizado para enlazar hojas de estilo.

    Ejemplo de uso:

    <link rel="stylesheet" href="estilos.css">

    Información adicional: El atributo rel define la relación entre el documento y el recurso, y el atributo href especifica la URL del recurso.


  6. <meta>

    Descripción: Proporciona metadatos sobre el documento, como la codificación de caracteres, la descripción, y las palabras clave.

    Ejemplo de uso:

    <meta charset="UTF-8">
<meta name="description" content="Descripción de la página">
<meta name="keywords" content="HTML, CSS, JavaScript">

    Información adicional: Los metadatos proporcionan información adicional a los navegadores y motores de búsqueda.


  7. <style>

    Descripción: Define estilos CSS para el documento. Los estilos se aplican a los elementos HTML dentro del documento.

    Ejemplo de uso:

    <style>
    body {
        font-family: Arial, sans-serif;
    }
    h1 {
        color: #333;
    }
</style>

    Información adicional: Los estilos dentro del elemento <style> afectan a todos los elementos HTML dentro del documento.


  8. <script>

    Descripción: Define un bloque de código JavaScript. Se puede colocar en el <head> o al final del <body>.

    Ejemplo de uso:

    <script>
    alert('Hola Mundo');
</script>

    Información adicional: Puedes usar el atributo src para cargar un archivo JavaScript externo.


  9. <body>

    Descripción: Define el contenido visible del documento HTML, incluyendo texto, imágenes, enlaces y otros elementos HTML.

    Ejemplo de uso:

    <body>
    <h1>Encabezado Principal</h1>
    <p>Este es un párrafo de texto.</p>
</body>

    Información adicional: El elemento <body> contiene todo el contenido que se muestra en la página web.


  10. <nav>

    Descripción: Define un bloque de navegación, comúnmente utilizado para contener enlaces a otras páginas o secciones del sitio web.

    Ejemplo de uso:

    <nav>
    <a href="index.html">Inicio</a>
    <a href="servicios.html">Servicios</a>
    <a href="contacto.html">Contacto</a>
</nav>

    Información adicional: El elemento <nav> ayuda a los motores de búsqueda y a los usuarios a encontrar fácilmente las secciones de navegación del sitio web.


  11. <main>

    Descripción: Define el contenido principal de un documento, excluyendo los encabezados, pies de página, y elementos de navegación. Debe aparecer solo una vez por documento.

    Ejemplo de uso:

    <main>
    <h1>Bienvenido a Nuestro Sitio</h1>
    <p>Aquí encontrarás la información más relevante y principal del sitio.</p>
</main>

    Información adicional: El elemento <main> mejora la accesibilidad y el SEO al identificar claramente el contenido principal del documento.


  12. <section>

    Descripción: Define una sección temática dentro de un documento. Puede incluir un encabezado y se utiliza para agrupar contenido relacionado.

    Ejemplo de uso:

    <section>
    <h2>Nuestro Equipo</h2>
    <p>Conoce a los miembros de nuestro equipo.</p>
</section>

    Información adicional: El elemento <section> es útil para organizar el contenido en partes lógicas y temáticas.


  13. <article>

    Descripción: Define un contenido independiente y autocontenible, como un artículo de blog, una noticia o una entrada de foro.

    Ejemplo de uso:

    <article>
    <h2>Título del Artículo</h2>
    <p>Este es el contenido del artículo. Aquí puedes poner texto, imágenes y otros elementos.</p>
</article>

    Información adicional: El elemento <article> se utiliza para encapsular contenido que puede ser distribuido o reutilizado de manera independiente.


  14. <aside>

    Descripción: Define un contenido relacionado que está separado del contenido principal, como una barra lateral, notas al margen o enlaces adicionales.

    Ejemplo de uso:

    <aside>
    <p>Este es un contenido adicional relacionado, como un enlace a una noticia relacionada o un bloque de información complementaria.</p>
</aside>

    Información adicional: El elemento <aside> se usa para información que está tangencialmente relacionada con el contenido principal pero que se puede leer por separado.


  15. <footer>

    Descripción: Define el pie de página de un documento o sección. Generalmente contiene información de contacto, derechos de autor, o enlaces de navegación.

    Ejemplo de uso:

    <footer>
    <p>© 2024 Mi Sitio Web</p>
    <a href="contacto.html">Contacto</a>
</footer>

    Información adicional: El elemento <footer> se utiliza para agrupar información que aparece al final de la página o sección.


  16. <header>

    Descripción: Define el encabezado de un documento o sección. Puede contener elementos como el logo, el título del sitio, y el menú de navegación.

    Ejemplo de uso:

    <header>
    <h1>Mi Sitio Web</h1>
    <nav>
        <a href="index.html">Inicio</a>
        <a href="acerca.html">Acerca de</a>
    </nav>
</header>

    Información adicional: El elemento <header> se utiliza para definir el contenido introductorio o de navegación de una página o sección.


  17. <h1>

    Descripción: Define el encabezado de nivel 1, utilizado para el título principal de la página.

    Ejemplo de uso:

    <h1>Título Principal</h1>

    Información adicional: Es el encabezado más importante y generalmente se usa una sola vez por página para el título principal.


  18. <h2>

    Descripción: Define el encabezado de nivel 2, utilizado para secciones secundarias del contenido.

    Ejemplo de uso:

    <h2>Subtítulo</h2>

    Información adicional: Es útil para dividir el contenido en secciones principales.


  19. <h3>

    Descripción: Define el encabezado de nivel 3, utilizado para subsecciones dentro de un encabezado de nivel 2.

    Ejemplo de uso:

    <h3>Subsección</h3>

    Información adicional: Ayuda a estructurar el contenido jerárquicamente.


  20. <h4>

    Descripción: Define el encabezado de nivel 4, utilizado para subsecciones dentro de un encabezado de nivel 3.

    Ejemplo de uso:

    <h4>Detalles</h4>

    Información adicional: Se usa para subdividir aún más el contenido si es necesario.


  21. <h5>

    Descripción: Define el encabezado de nivel 5, utilizado para subsecciones dentro de un encabezado de nivel 4.

    Ejemplo de uso:

    <h5>Más Detalles</h5>

    Información adicional: Es menos común, pero útil para una estructura de contenido más granular.


  22. <h6>

    Descripción: Define el encabezado de nivel 6, utilizado para subsecciones dentro de un encabezado de nivel 5.

    Ejemplo de uso:

    <h6>El Más Bajo Nivel</h6>

    Información adicional: Es el encabezado menos importante y se usa raramente, pero puede ser útil para detalles menores.


  23. <p>

    Descripción: Define un párrafo de texto.

    Ejemplo de uso:

    <p>Este es un párrafo de ejemplo.</p>

    Información adicional: Los párrafos se separan automáticamente con un espacio en blanco antes y después.


  24. <hr>

    Descripción: Inserta una línea horizontal, a menudo utilizada para separar contenido.

    Ejemplo de uso:

    <hr>

    Información adicional: No tiene etiqueta de cierre.


  25. <pre>

    Descripción: Define texto preformateado, que conserva los espacios y saltos de línea.

    Ejemplo de uso:

    <pre>
        Texto con
        Espacios y
        Saltos de línea
    </pre>

    Información adicional: Útil para mostrar código o texto con formato específico.


  26. <blockquote>

    Descripción: Define una cita de bloque, generalmente usada para citas extensas.

    Ejemplo de uso:

    <blockquote>Cita de ejemplo</blockquote>

    Información adicional: Generalmente se usa con CSS para aplicar sangrías o estilos especiales.


  27. <ul>

    Descripción: Define una lista desordenada.

    Ejemplo de uso:

    <ul>
                    <li>Elemento 1</li>
                    <li>Elemento 2</li>
                    </ul>

    Información adicional: Los elementos de la lista se definen con la etiqueta <li>.


  28. <ol>

    Descripción: Define una lista ordenada.

    Ejemplo de uso:

    <ol>
                    <li>Primer elemento</li>
                    <li>Segundo elemento</li>
                    </ol>

    Información adicional: Los elementos de la lista se definen con la etiqueta <li>.


  29. <li>

    Descripción: Define un elemento de una lista. Se utiliza dentro de elementos de lista como <ul> (lista desordenada) o <ol> (lista ordenada).

    Ejemplo de uso:

    <ul>
                    <li>Elemento de lista 1</li>
                    <li>Elemento de lista 2</li>
                </ul>

    Información adicional: El elemento <li> se usa para listar elementos en un formato ordenado o desordenado y puede contener otros elementos HTML, como texto, imágenes o enlaces.


  30. <dl>

    Descripción: Define una lista de definiciones.

    Ejemplo de uso:

    <dl>
        <dt>Término</dt>
        <dd>Definición del término</dd>
    </dl>

    Información adicional: Usa <dt> para términos y <dd> para definiciones.


  31. <dt>

    Descripción: Define un término en una lista de definiciones.

    Ejemplo de uso:

    <dl>
        <dt>Término</dt>
        <dd>Definición del término</dd>
    </dl>

    Información adicional: Se usa dentro de una lista <dl>.


  32. <dd>

    Descripción: Define una definición o descripción en una lista de definiciones.

    Ejemplo de uso:

    <dl>
        <dt>Término</dt>
        <dd>Definición del término</dd>
    </dl>

    Información adicional: Se usa para proporcionar información adicional sobre el término definido.


  33. <figure>

    Descripción: Define un contenido de medios que puede ser independiente del contenido principal, como imágenes, gráficos, o videos. Generalmente se usa junto con <figcaption> para proporcionar una leyenda.

    Ejemplo de uso:

    <figure>
    <img src="imagen.jpg" alt="Descripción de la imagen">
    <figcaption>Esta es una imagen de ejemplo.</figcaption>
</figure>

    Información adicional: El elemento <figure> agrupa el contenido relacionado, haciendo que sea más fácil agregar una leyenda o descripción y mantener la asociación entre la imagen y su descripción.


  34. <figcaption>

    Descripción: Define una leyenda o descripción para el contenido de un elemento <figure>. Proporciona información adicional sobre el contenido representado en el <figure>.

    Ejemplo de uso:

    <figure>
    <img src="grafico.jpg" alt="Gráfico de ventas">
    <figcaption>Gráfico que muestra las ventas trimestrales.</figcaption>
</figure>

    Información adicional: La etiqueta <figcaption> debe ser el primer o el último elemento dentro del elemento <figure>, y proporciona contexto o explicación adicional al contenido del <figure>.


  35. <div>

    Descripción: Define una sección o contenedor genérico en el documento.

    Ejemplo de uso:

    <div>Contenido del contenedor</div>

    Información adicional: Generalmente se usa junto con CSS para diseño y estilo.


  36. <a>

    Descripción: Define un hipervínculo. El atributo href especifica la URL del destino del enlace.

    Ejemplo de uso:

    <a href="https://www.ejemplo.com">Visita Ejemplo</a>

    Información adicional: Puedes usar atributos adicionales como target para definir cómo se abrirá el enlace.


  37. <b>

    Descripción: Define un texto en negrita. Se usa para enfatizar el texto, pero no tiene un significado semántico adicional en HTML5.

    Ejemplo de uso:

    <p>Este es un texto en <b>negrita</b>.</p>

    Información adicional: La etiqueta <b> no debería usarse para enfatizar el texto. En su lugar, se recomienda usar <strong> para un énfasis semántico.


  38. <i>

    Descripción: Define un texto en cursiva. Se usa para representar texto en un estilo de cursiva, sin tener un significado semántico adicional en HTML5.

    Ejemplo de uso:

    <p>Este es un texto en <i>cursiva</i>.</p>

    Información adicional: La etiqueta <i> no debería usarse para resaltar o enfatizar el texto. En su lugar, se recomienda usar <em> para un énfasis semántico.


  39. <u>

    Descripción: Define un texto subrayado. Se usa para representar texto con una línea horizontal debajo, sin un significado semántico específico en HTML5.

    Ejemplo de uso:

    <p>Este es un texto <u>subrayado</u>.</p>

    Información adicional: La etiqueta <u> se usa principalmente para texto que debe ser subrayado, pero no tiene un significado semántico. Para enlaces, usa <a> con estilo de subrayado.


  40. <strong>

    Descripción: Define texto con énfasis fuerte, generalmente renderizado en negrita.

    Ejemplo de uso:

    <strong>Texto en negrita</strong>

    Información adicional: Se utiliza para destacar partes importantes del texto.


  41. <small>

    Descripción: Define texto con un tamaño de fuente más pequeño que el texto circundante. Se usa para indicar texto secundario o de menor importancia.

    Ejemplo de uso:

    <p>Este es un texto normal y este es un texto <small>más pequeño</small>.</p>

    Información adicional: El tamaño de la fuente dentro de <small> puede variar dependiendo del estilo CSS aplicado al documento.


  42. <sub>

    Descripción: Define texto que debe aparecer en subíndice. Se usa comúnmente en fórmulas químicas o matemáticas.

    Ejemplo de uso:

    <p>El agua se representa como H<sub>2</sub>O.</p>

    Información adicional: El texto dentro de <sub> se muestra en una posición más baja que el texto normal, y con un tamaño de fuente más pequeño.


  43. <sup>

    Descripción: Define texto que debe aparecer en superíndice. Se usa comúnmente en exponentes matemáticos o en notaciones científicas.

    Ejemplo de uso:

    <p>El cuadrado de 2 se representa como 2<sup>2</sup>.</p>

    Información adicional: El texto dentro de <sup> se muestra en una posición más alta que el texto normal, y con un tamaño de fuente más pequeño.


  44. <mark>

    Descripción: Define texto que debe ser resaltado, generalmente con un fondo amarillo. Se usa para destacar partes importantes del contenido.

    Ejemplo de uso:

    <p>Este es un texto <mark>resaltado</mark> para destacar su importancia.</p>

    Información adicional: El resaltado dentro de <mark> ayuda a llamar la atención sobre partes específicas del texto, como en búsquedas o resultados.


  45. <cite>

    Descripción: Define el título de una obra o referencia.

    Ejemplo de uso:

    <cite>Título de la obra</cite>

    Información adicional: Se usa para indicar la fuente de una cita o referencia.


  46. <span>

    Descripción: Define una sección o contenedor en línea, útil para aplicar estilos o manipular parte del texto.

    Ejemplo de uso:

    <span>Texto en línea</span>

    Información adicional: A menudo se usa junto con CSS para aplicar estilos a una porción específica del contenido.


  47. <br>

    Descripción: Inserta un salto de línea en el texto.

    Ejemplo de uso:

    Primera línea<br>Segunda línea

    Información adicional: No tiene etiqueta de cierre.


  48. <img>

    Descripción: Inserta una imagen en la página. El atributo src especifica la URL de la imagen.

    Ejemplo de uso:

    <img src="imagen.jpg" alt="Descripción de la imagen">

    Información adicional: Puedes usar atributos como width y height para ajustar el tamaño de la imagen.


  49. <canvas>

    Descripción: Define un área en la que se puede dibujar gráficos mediante scripting, como JavaScript. Se utiliza para crear gráficos, animaciones, y otras imágenes dinámicas.

    Ejemplo de uso:

    <canvas id="miCanvas" width="200" height="200">Su navegador no soporta el elemento <canvas>.</canvas>
<script>
    var canvas = document.getElementById('miCanvas');
    var context = canvas.getContext('2d');
    context.fillStyle = 'blue';
    context.fillRect(10, 10, 150, 100);
</script>

    Información adicional: El contexto del lienzo se obtiene mediante getContext('2d') para gráficos 2D o getContext('webgl') para gráficos 3D. El lienzo se utiliza con JavaScript para dibujar y manipular gráficos dinámicamente.


  50. <iframe>

    Descripción: Define un marco embebido que puede mostrar otra página HTML dentro de la página actual.

    Ejemplo de uso:

    <iframe src="https://www.ejemplo.com" width="600" height="400"></iframe>

    Información adicional: Puedes usar atributos como width y height para ajustar el tamaño del marco.

  51. <embed>

    Descripción: Define un contenido externo embebido, como un archivo multimedia o documento.

    Ejemplo de uso:

    <embed src="video.mp4" type="video/mp4" width="640" height="480">

    Información adicional: A menudo se usa para integrar archivos multimedia directamente en la página.


  52. <object>

    Descripción: Define un objeto embebido, que puede ser un archivo multimedia, applet o documento.

    Ejemplo de uso:

    <object data="documento.pdf" type="application/pdf" width="600" height="400"></object>

    Información adicional: Utiliza el atributo data para especificar la fuente del objeto.


  53. <audio>

    Descripción: Define contenido de audio. Puedes usar atributos para controlar la reproducción.

    Ejemplo de uso:

    <audio controls>
<source src="audio.mp3" type="audio/mpeg">
Su navegador no soporta el elemento audio.
</audio>

    Información adicional: El atributo controls muestra los controles de reproducción.


  54. <video>

    Descripción: Define contenido de video. Puedes especificar múltiples fuentes y usar atributos para controlar la reproducción.

    Ejemplo de uso:

    <video width="640" height="480" controls>
<source src="video.mp4" type="video/mp4">
Su navegador no soporta el elemento video.
</video>

    Información adicional: Puedes incluir múltiples fuentes dentro del elemento <video> para compatibilidad con diferentes formatos.


  55. <source>

    Descripción: Define múltiples fuentes para elementos <video> o <audio>, permitiendo la selección de diferentes formatos.

    Ejemplo de uso:

    <video controls>
<source src="video.mp4" type="video/mp4">
<source src="video.ogv" type="video/ogg">
Su navegador no soporta el elemento video.
</video>

    Información adicional: Los navegadores seleccionarán automáticamente el primer formato soportado.


  56. <svg>

    Descripción: Define un contenedor para gráficos vectoriales escalables (SVG) en el documento HTML. Permite dibujar gráficos, formas, y otros elementos gráficos utilizando una sintaxis basada en XML.

    Ejemplo de uso:

    <svg width="100" height="100" xmlns="http://www.w3.org/2000/svg">
        <circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red" />
        <rect x="10" y="10" width="30" height="30" stroke="black" stroke-width="2" fill="blue" />
        <line x1="10" y1="80" x2="90" y2="80" stroke="black" stroke-width="2" />
    </svg>

    Información adicional: El elemento <svg> puede contener varios elementos gráficos como <circle>, <rect>, y <line>. Los gráficos SVG son escalables, lo que significa que se pueden redimensionar sin perder calidad.


  57. <table>

    Descripción: Define una tabla.

    Ejemplo de uso:

    <table>
    <tr>
        <th>Encabezado 1</th>
        <th>Encabezado 2</th>
    </tr>
    <tr>
        <td>Celda 1</td>
        <td>Celda 2</td>
    </tr>
    </table>

    Información adicional: Usa <tr> para filas, <th> para encabezados y <td> para celdas.


  58. <caption>

    Descripción: Define un título para una tabla. Se coloca dentro del elemento <table>.

    Ejemplo de uso:

    <table>
    <caption>Título de la Tabla</caption>
    <tr>
        <th>Encabezado 1</th>
        <th>Encabezado 2</th>
    </tr>
    <tr>
        <td>Dato 1</td>
        <td>Dato 2</td>
    </tr>
</table>

    Información adicional: El elemento <caption> es opcional y se usa para proporcionar un título o descripción a la tabla.


  59. <colgroup>

    Descripción: Agrupa uno o más elementos <col> para aplicar propiedades de estilo a columnas específicas en una tabla.

    Ejemplo de uso:

    <table>
    <colgroup>
        <col style="background-color: #f2f2f2">
        <col style="background-color: #e6e6e6">
    </colgroup>
    <tr>
        <th>Encabezado 1</th>
        <th>Encabezado 2</th>
    </tr>
    <tr>
        <td>Dato 1</td>
        <td>Dato 2</td>
    </tr>
</table>

    Información adicional: El elemento <colgroup> se utiliza para aplicar estilos o atributos a grupos de columnas en una tabla.


  60. <tbody>

    Descripción: Define un cuerpo de una tabla en HTML, agrupando filas de datos.

    Ejemplo de uso:

    <table>
        <thead><tr><th>Encabezado</th></tr></thead>
        <tbody>
            <tr><td>Dato 1</td></tr>
            <tr><td>Dato 2</td></tr>
        </tbody>
    </table>

    Información adicional: Agrupa las filas de datos en una tabla, separándolas de los encabezados y el pie de página.


  61. <thead>

    Descripción: Define un grupo de encabezados en una tabla HTML.

    Ejemplo de uso:

    <table>
        <thead>
            <tr>
                <th>Encabezado 1</th>
                <th>Encabezado 2</th>
            </tr>
        </thead>
        <tbody><!-- Filas de datos --></tbody>
    </table>

    Información adicional: Se usa para agrupar los encabezados de la tabla y mejorar la legibilidad y el diseño.


  62. <tfoot>

    Descripción: Define un grupo de pie de tabla en HTML, generalmente usado para sumarizar o proporcionar información adicional.

    Ejemplo de uso:

    <table>
        <tfoot>
            <tr>
                <td colspan="2">Total:</td>
                <td>$100</td>
            </tr>
        </tfoot>
        <tbody><!-- Filas de datos --></tbody>
    </table>

    Información adicional: Es útil para proporcionar sumarios, totales o notas al pie de la tabla.


  63. <tr>

    Descripción: Define una fila en una tabla. Se utiliza dentro de los elementos <table>, <thead>, <tbody> y <tfoot>.

    Ejemplo de uso:

    <table>
    <tr>
        <td>Celda 1</td>
        <td>Celda 2</td>
    </tr>
</table>

    Información adicional: Cada fila en la tabla se define con un elemento <tr>, y dentro de cada fila puedes colocar celdas con <td> o <th>.


  64. <td>

    Descripción: Define una celda en una fila de una tabla. Se utiliza dentro del elemento <tr>.

    Ejemplo de uso:

    <table>
    <tr>
        <td>Celda 1</td>
        <td>Celda 2</td>
    </tr>
</table>

    Información adicional: El elemento <td> representa una celda en una tabla que puede contener texto, imágenes, enlaces u otros elementos HTML.


  65. <th>

    Descripción: Define una celda de encabezado en una tabla. Se utiliza dentro del elemento <tr> y el texto se muestra en negrita y centrado por defecto.

    Ejemplo de uso:

    <table>
    <tr>
        <th>Encabezado 1</th>
        <th>Encabezado 2</th>
    </tr>
    <tr>
        <td>Celda 1</td>
        <td>Celda 2</td>
    </tr>
</table>

    Información adicional: El elemento <th> se utiliza para definir encabezados de columnas o filas en una tabla, proporcionando una forma clara de organizar la información.


  66. <fieldset>

    Descripción: Agrupa controles relacionados dentro de un formulario.

        <legend>Información Personal</legend>

        Edad: <input type="number">

  67. <legend>

    Descripción: Define una leyenda para un grupo de controles en un formulario, dentro de un <fieldset>.

    Ejemplo de uso:

    <fieldset>
        <legend>Información Personal</legend>
        Nombre: <input type="text">
    
        Edad: <input type="number">
    </fieldset>

    Información adicional: Proporciona un título o descripción para el contenido del <fieldset>.


  68. <label>

    Descripción: Define una etiqueta para un control de formulario.

    Ejemplo de uso:

    <label for="nombre">Nombre:</label>
    <input type="text" id="nombre" name="nombre">

    Información adicional: El atributo for asocia la etiqueta con el control de formulario correspondiente.


  69. <input>

    Descripción: Define un campo de entrada en un formulario.

    Ejemplo de uso:

    <input type="text" name="nombre">

    Información adicional: El atributo type define el tipo de campo (texto, contraseña, etc.).


  70. <button>

    Descripción: Define un botón que se puede hacer clic.

    Ejemplo de uso:

    <button type="button">Haz clic</button>

    Información adicional: Puedes usar el atributo type para definir el comportamiento del botón.


  71. <select>

    Descripción: Define un menú desplegable.

    Ejemplo de uso:

    <select name="opcion">
        <option value="1">Opción 1</option>
        <option value="2">Opción 2</option>
        </select>

    Información adicional: Los elementos del menú se definen con la etiqueta <option>.


  72. <option>

    Descripción: Define una opción en un menú desplegable dentro de un elemento <select>. Cada <option> representa una opción individual que el usuario puede seleccionar.

    Ejemplo de uso:

    <select>
    <option value="1">Opción 1</option>
    <option value="2">Opción 2</option>
    <option value="3">Opción 3</option>
</select>

    Información adicional: El atributo value del <option> especifica el valor que se enviará al servidor cuando se envíe el formulario. También puedes usar el atributo selected para preseleccionar una opción.


  73. <textarea>

    Descripción: Define un área de texto multilínea.

    Ejemplo de uso:

    <textarea name="comentario">Escribe aquí</textarea>

    Información adicional: Puedes usar atributos como rows y cols para ajustar el tamaño del área de texto.


  74. <form>

    Descripción: Define un formulario para la entrada del usuario.

    Ejemplo de uso:

    <form action="/enviar" method="post">
    <input type="text" name="nombre">
    <input type="submit" value="Enviar">
    </form>

    Información adicional: Usa el atributo action para definir a dónde se envía el formulario.


  75. <label>

    Descripción: Define una etiqueta para un elemento de formulario.

    Ejemplo de uso:

    <label for="nombre">Nombre:</label>
    <input type="text" id="nombre" name="nombre">

  76. <em>

    Descripción: Define texto con énfasis, generalmente renderizado en cursiva.

    Ejemplo de uso:

    <em>Texto en cursiva</em>

    Información adicional: Se usa para enfatizar palabras o frases en el texto.


  77. <code>

    Descripción: Define un fragmento de código.

    Ejemplo de uso:

    <code>Código de ejemplo</code>

    Información adicional: Usado para mostrar texto en formato monoespaciado.


  78. <abbr>

    Descripción: Define una abreviatura o acrónimo. El atributo title proporciona la expansión.

    Ejemplo de uso:

    <abbr title="HyperText Markup Language">HTML</abbr>

    Información adicional: El atributo title muestra el texto completo al pasar el cursor sobre la abreviatura.


  79. <address>

    Ejemplo de uso:

</address>

    Información adicional: Generalmente se usa para mostrar detalles de contacto.


  80. <details>

    Descripción: Define un control de detalles que puede ser abierto o cerrado por el usuario.

    <details>
    <summary>Más información</summary>
    Detalles adicionales que se muestran cuando el usuario expande el contenido.
</details>

    Información adicional: Utiliza el elemento <summary> para proporcionar un título que se puede hacer clic para expandir o contraer el contenido.


  81. <summary>

    Descripción: Define el título visible de un elemento <details>, que puede ser clicado para mostrar u ocultar el contenido.

    Ejemplo de uso:

    <details>
    <summary>Resumen de detalles</summary>
    Información adicional que se muestra al expandir el resumen.
</details>

  82. <form>

    Descripción: Define un formulario para la entrada del usuario.

    <form action="/enviar" method="post">
    Nombre: <input type="text" name="nombre">
    
    <input type="submit" value="Enviar">
</form>

  83. <input>

    Descripción: Define un campo de entrada en un formulario.

    Ejemplo de uso:

    <input type="text" name="usuario" placeholder="Introduce tu nombre">

    Información adicional: El atributo type define el tipo de entrada, como texto, botón, etc.


  84. <button>

    Descripción: Define un botón interactivo en un formulario o interfaz.

    Ejemplo de uso:

    <button type="submit">Enviar</button>

    Información adicional: Puedes usar el atributo type para especificar el tipo de botón (submit, button, reset).


  85. <bdo>

    Descripción: Define el texto que debe mostrarse en un orden de dirección específico, como de derecha a izquierda o viceversa. Es útil para cambiar la dirección del texto en documentos multilingües.

    Ejemplo de uso:

    <p>Texto normal, <bdo dir="rtl">texto en dirección de derecha a izquierda</bdo>.</p>

    Información adicional: El atributo dir puede tomar valores como "ltr" (de izquierda a derecha) y "rtl" (de derecha a izquierda).


  86. <q>

    Descripción: Define una cita en línea corta. El navegador suele agregar automáticamente comillas alrededor del contenido.

    Ejemplo de uso:

    <p>Ella dijo: <q>Hola, ¿cómo estás?</q></p>

    Información adicional: La etiqueta <q> es para citas cortas. Para citas más largas, utiliza <blockquote>.


  87. <ins>

    Descripción: Define el contenido que se ha insertado en el documento. Generalmente se utiliza con una línea debajo para indicar que se ha agregado nuevo contenido.

    Ejemplo de uso:

    <p>Texto original. <ins>Texto añadido.</ins> Texto adicional.</p>

    Información adicional: Se puede utilizar el atributo datetime para especificar la fecha y hora en que se realizó la inserción.


  88. <del>

    Descripción: Define el contenido que se ha eliminado del documento. Usualmente se muestra con una línea en el medio para indicar que se ha borrado.

    Ejemplo de uso:

    <p>Texto original. <del>Texto eliminado.</del> Texto adicional.</p>

    Información adicional: El atributo datetime también se puede usar para especificar la fecha y hora en que se eliminó el contenido.


  89. <dfn>

    Descripción: Define una definición de un término. Se usa en conjunto con una definición de término o para destacar un término que está siendo definido.

    Ejemplo de uso:

    <p>El término <dfn>HTML</dfn> significa HyperText Markup Language.</p>

    Información adicional: El elemento <dfn> generalmente se usa en el contexto de definir términos técnicos o especializados en un documento.


  90. <kbd>

    Información adicional: El elemento <kbd> se utiliza para representar texto que el usuario debe ingresar desde un teclado.


  91. <samp>

    Información adicional: El texto dentro de <samp> se muestra con un estilo monoespaciado, similar al que se utiliza en la salida de la consola.


  92. <var>

    Descripción: Define una variable en un contexto matemático o de programación. Se utiliza para representar variables en expresiones o fórmulas.

    Ejemplo de uso:

    <p>En la fórmula <var>x</var> + <var>y</var> = <var>z</var>, <var>x</var> y <var>y</var> son variables.</p>

    Información adicional: El elemento <var> es útil para representar variables en expresiones matemáticas o en ejemplos de código.


  93. <base>

    Descripción: Define la URL base para todas las URLs relativas en un documento. Debe estar dentro del elemento <head>.

</head>
<body>
    <a href="pagina.html">Enlace a la página</a>
</body>

    Información adicional: La etiqueta <base> debe ser la primera en el <head> y establece una base para todas las URLs relativas en el documento.


  94. <area>

    Descripción: Define una región activa dentro de un mapa de imagen. Se usa dentro de un elemento <map> para definir áreas clicables en una imagen.

    Ejemplo de uso:

    <map name="mapaEjemplo">
    <area shape="rect" coords="34,44,270,350" href="pagina.html" alt="Descripción del área">
</map>

    Información adicional: El atributo shape define la forma del área (por ejemplo, rect para rectángulo, circle para círculo), y coords define las coordenadas de la forma.


  95. <map>

    Descripción: Define un mapa de imagen, que es una imagen con áreas clicables. Se usa junto con <area> para crear enlaces en diferentes partes de la imagen.

    Ejemplo de uso:

    <img src="imagen.jpg" usemap="#mapaEjemplo" alt="Imagen con mapa">
<map name="mapaEjemplo">
    <area shape="rect" coords="34,44,270,350" href="pagina.html" alt="Descripción del área">
</map>

    Información adicional: El atributo usemap en la imagen se usa para asociar la imagen con el mapa definido.


  96. <param>

    Información adicional: Los parámetros definidos con <param> son utilizados por el objeto embebido para configurar su comportamiento o apariencia.

    <!-- Este es un comentario en HTML -->

    Información adicional: Los comentarios pueden contener cualquier texto, pero no se deben usar para ocultar contenido HTML ya que puede ser visible para usuarios con herramientas de desarrollo.

    Descripción: Agrupa varios elementos de encabezado (`<h1>`, `<h2>`, `<h3>`, etc.) en una única unidad, de manera que se traten como un solo bloque para el propósito de la jerarquía de encabezados.

    <h2>Subtítulo</h2>
</hgroup>

    Información adicional: El elemento <hgroup> se ha vuelto obsoleto en HTML5 y se recomienda usar `<header>` en su lugar para agrupar encabezados.


  99. <s>

    Descripción: Define texto con una línea a través de él, generalmente usado para indicar que el texto está obsoleto o ha sido reemplazado.

    Ejemplo de uso:

    <p>Texto original con <s>texto tachado</s> tachado.</p>

    Información adicional: La etiqueta <s> se usa para representar texto que ya no es relevante o que ha sido reemplazado por otra cosa. Para eliminar texto con significado semántico, usa <del>.


  100. <time>

    Descripción: Define una fecha y/o hora en el documento. Facilita la representación y el manejo de fechas y horas en el formato de datos.

    Ejemplo de uso:

    <time datetime="2024-09-11T14:00">11 de septiembre de 2024, 14:00</time>

    Información adicional: El atributo datetime especifica la fecha y hora en formato de fecha y hora ISO 8601. Esto facilita la integración con otras aplicaciones y servicios.


  101. <track>

    Ejemplo de uso:

    Información adicional: El atributo kind puede ser "subtitles", "captions", "descriptions", o "chapters". El atributo srclang define el idioma de la pista.


  102. <wbr>

    Ejemplo de uso:

    <p>Texto muy largo que podría necesitar un <wbr>quebrantamiento</wbr> en la línea.</p>

    Información adicional: La etiqueta <wbr> es una sugerencia para los navegadores sobre dónde permitir el salto de línea. No garantiza que se produzca un salto de línea, pero proporciona una guía.
